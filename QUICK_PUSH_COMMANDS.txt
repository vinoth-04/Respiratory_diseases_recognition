# Quick Command Reference for GitHub Push

Copy and paste these commands in order:

## FIRST TIME SETUP (Run once)

```powershell
# Navigate to project
cd "d:\vinoth_projects_25\machine learning projects\lung disease prediction\lung_predicition"

# Configure git (only first time)
git config --global user.name "Vinoth"
git config --global user.email "your.email@example.com"

# Check git status
git status
```

## PUSHING TO GITHUB

### Option 1: Complete Fresh Push (Recommended for first time)

```powershell
# Initialize git
git init

# Add all files
git add .

# Commit with message
git commit -m "Initial commit: Clean project structure with ML models"

# Set main branch
git branch -M main

# Add remote (if not already added)
git remote add origin https://github.com/vinoth-04/Respiratory_diseases_recognition.git

# Push to GitHub
git push -u origin main
```

### Option 2: Normal Update (After initial push)

```powershell
# Check what changed
git status

# Add changes
git add .

# Commit
git commit -m "Update: Description of changes"

# Push
git push origin main
```

### Option 3: Clean Rewrite (If previous commits are messy)

```powershell
# Remove old git history
Remove-Item .git -Recurse -Force

# Start fresh
git init
git add .
git commit -m "Initial commit: Clean project structure"
git branch -M main
git remote add origin https://github.com/vinoth-04/Respiratory_diseases_recognition.git
git push -u origin --force main
```

## VERIFICATION

```powershell
# Check current branch
git branch

# Check remote
git remote -v

# View commit history
git log --oneline
```

## COMMON ISSUES & FIXES

```powershell
# If "fatal: not a git repository"
git init

# If files not added
git add .

# If remote already exists
git remote set-url origin https://github.com/vinoth-04/Respiratory_diseases_recognition.git

# If push fails - pull first
git pull origin main
git push origin main

# If authentication fails
git credential reject
git push (you'll be prompted for token)
```

## ONE-LINER (Full Setup)

```powershell
cd "d:\vinoth_projects_25\machine learning projects\lung disease prediction\lung_predicition"; git init; git add .; git commit -m "Initial commit: Clean project structure"; git branch -M main; git remote add origin https://github.com/vinoth-04/Respiratory_diseases_recognition.git; git push -u origin main
```
